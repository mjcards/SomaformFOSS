# SomaForm Roadmap

## Now (v0.1 — current)
- Video/image upload + Claude vision analysis
- Maglischo framework prompt engine
- Phase scoring: Approach, Into Wall, Turn, Push-Off, Breakout
- All four strokes + auto-detect
- Swimmer-facing display mode (large scores, key cue)
- Coach report view

## Next (v0.2)
- Live camera feed (no upload required — poolside use)
- Automatic swimmer detection (analysis triggers on presence)
- Persistent session log (browser storage)
- Shareable results link

## Soon (v0.3)
- Lane-line calibration for distance/speed estimation
- Stroke count and stroke rate calculation
- Turn time measurement (touch to push)
- Breakout distance measurement

## Later
- On-device analysis (no API key required)
- Multi-lane support
- Club admin dashboard
- Swimmer profiles and progress tracking over time
- Export to PDF coaching report

## Community Wishlist
Open an issue tagged `roadmap` to suggest features.
